☁️ Sincronizar entre computadores (GitHub Gist)

Para usar a tua coleção em vários computadores, configura aqui um token do GitHub. Os teus dados ficam guardados num Gist privado e sincronizados em todos os dispositivos onde uses o mesmo token e Gist ID.

Como configurar (passo a passo)
  1. Vai a github.com/settings/tokens e cria um Personal Access Token (classic) com permissão gist.
  2. Cola o token aqui em baixo e clica em "Guardar Token".
  3. Na primeira vez, clica em "Criar Gist Novo" — vai gerar um Gist ID. Copia esse ID.
  4. No outro computador, instala o site, cola o mesmo token e o mesmo Gist ID, e clica em "Carregar do Gist".
